# Guía de instalación de Astro

![Logo de Astro](img/logo.png){ width="220" }

Bienvenido a la **guía de instalación de Astro**. Aquí encontrarás los pasos
necesarios para instalar Astro, crear tu primer proyecto, verificar que funcione
correctamente y resolver los problemas más comunes.

## ¿Qué es Astro?

Astro es un **framework web de código abierto** para construir sitios rápidos y
centrados en el contenido (blogs, documentación, portafolios, sitios de marketing).
Su característica clave es la **arquitectura de islas**: por defecto envía cero
JavaScript al navegador y solo hidrata los componentes interactivos que realmente
lo necesitan, lo que produce páginas muy ligeras.

!!! note "Antes de empezar"
    Esta guía asume conocimientos básicos de la línea de comandos y de Node.js.
    No necesitas experiencia previa con Astro.
